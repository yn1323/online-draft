## 1. アプリケーション概要

### プロダクトビジョン
「友人同士が音声通話しながら、URLシェアだけで手軽に参加できるドラフトゲーム体験を提供し、どんなジャンルでも盛り上がれる汎用プラットフォームを目指す」

### コンセプト
**「推しを選んで、かぶったら勝負！」** - リアルタイムマルチプレイヤードラフトゲーム

### 解決する課題

| 課題 | 詳細 |
|------|------|
| 登録が面倒 | 既存ドラフトゲームはアカウント作成必須 |
| 汎用性がない | POG専用、野球専用など特定ジャンル限定 |
| 場所の制約 | オフライン開催は集まれないと無理 |
| 盛り上がりに欠ける | リアルタイム性や演出がない |

## 2. 主要機能一覧

### A. グループ・ルーム管理
- ルーム作成（グループ名入力のみで簡単作成）
- URLシェア・QRコード共有による参加招待
- ルーム参加履歴の保存

### B. ユーザー管理
- Firebase Anonymous Authentication（匿名認証）
- 18種類の動物アイコンから選択できるアバター機能
- 複数デバイスでの同一ユーザー使用対応

### C. ドラフト機能（コア機能）
- **一斉入力方式** - 従来の順番制ではなく全員が同時選択
- **複数ラウンド対応** - 複数巡目で連続実施
- **アイテム選択 + コメント機能** - 理由や予想をコメント付きで入力
- **重複指名処理** - 同じアイテムを選んだらランダム抽選で勝者決定
- **敗者再選択** - 重複指名敗者は同一巡目内で別選択肢を選択可能
- **編集機能** - 過去ラウンドの選択を後から編集可能

### D. ゲーム演出
- **開票演出** - スロット風の一斉開票演出（3秒のアニメーション）
- **成功/失敗表示** - アイテム獲得成功・失敗の視覚的フィードバック

### E. リアルタイムチャット
- グループチャット（参加者全員でワイワイ）
- 新着メッセージの自動スクロール
- システムログ（編集履歴・獲得ログ・巡目完了ログ等）

### F. UI/UX
- レスポンシブ対応（PC・スマホ両対応）
- ダークモード対応
- わかりやすいUI設計

## 4. 競合サービス比較

### 競合一覧

| サービス | 特徴 | URL |
|----------|------|-----|
| POG Draft Meeting | POG専用、順番制ドラフト、自動抽選機能 | https://pog-draft-meeting.vercel.app/ |
| みんなのPOG (netkeiba) | POG専用、競馬DB連携、最大50名対応 | https://pog.netkeiba.com/ |
| プロ野球#LIVE | NPB公認ファンタジー、14名ドラフト | https://pacificleague.com/ |
| JRA-VAN POG | POG専用、公式データ連携 | https://jra-van.jp/pog/ |
| スプレッドシート+LINE | 手動管理、あみだくじで抽選 | - |

### 競合の特徴詳細

#### POG Draft Meeting
- スプレッドシートやLINEでのやり取り不要
- 指名管理からドラフト進行まで一括管理
- 競合時は自動抽選機能
- レスポンシブ対応

#### netkeiba「みんなのPOG」
- POGをweb上で管理、全て無料
- レース情報は自動更新
- ポイントや順位は自動集計
- 1グループ最大50名、1人50頭まで登録可能
- 中央競馬・地方競馬対応

#### プロ野球#LIVE（NPB公認）
- 実際のプロ野球成績と連動
- 野手9名、投手5名の計14名をドラフト
- 競合時は抽選で決定
- オンライン通話・チャット可能

### OnlineDraftとの比較表

| 強み | OnlineDraft | 競合 |
|------|-------------|------|
| **汎用性** | ✅ 何でもOK | ❌ ジャンル限定 |
| **一斉入力** | ✅ 順番待ちなし | ❌ 順番制が多い |
| **演出** | ✅ スロット風開票 | ❌ 淡々と進行 |
| **登録** | ✅ 不要（匿名OK） | ❌ アカウント必須が多い |
| **複数ツール不要** | ✅ 1つで完結 | ❌ スプシ+LINE等併用 |
| **スマホ対応** | ✅ 完全レスポンシブ | △ PC優先が多い |

## 5. ターゲットユーザー（ペルソナ）

### 対象ユーザー層

| ペルソナ | 詳細 | 利用シーン |
|----------|------|------------|
| **スポーツファングループ** | 野球・サッカーファンの友人グループ | シーズン開始前、ドラフト会議後 |
| **アニメ・ゲームオタク仲間** | 今期アニメ・ソシャゲ好き | 新クール開始時、新作発売時 |
| **職場・サークルの軽いノリ** | 忘年会の余興、昼休みの遊び | 年末年始、イベント前 |
| **ネットコミュニティ** | Discordサーバー、配信者企画 | オンラインイベント時 |

### ユーザー属性

| 属性 | 詳細 |
|------|------|
| 年齢層 | ミレニアル世代・Z世代（20-40代中心） |
| デバイス | スマートフォンメイン、PCサブ |
| 参加人数 | 2-8人（典型的には4-8人） |
| プレイ時間 | 1ゲーム10-15分 |

### 利用シーン詳細

#### シナリオ1: 野球ファングループのドラフト
- **参加者**: 6人の野球ファン友人グループ
- **対象**: 2024年プロ野球選手
- **時期**: シーズン開始前
- **フロー**:
  1. 幹事がルーム作成「2024プロ野球ドラフト」
  2. LINEグループにURL共有
  3. 各自が好きなタイミングで参加、アバター選択
  4. 全員揃ったらドラフト開始（5巡目まで実施）
  5. チャットで「その選手狙ってた〜」等でワイワイ
  6. シーズン中に振り返って盛り上がり

#### シナリオ2: アニメファンのキャラドラフト
- **参加者**: 4人のアニメ好き
- **対象**: 今期アニメキャラクター
- **時期**: 新アニメクール開始時
- **フロー**:
  1. Discord通話しながらドラフト実施
  2. 各自が推しキャラをコメント付きで選択
  3. 「このキャラ絶対活躍する！」等の予想コメント
  4. 次クールに結果を振り返り

#### シナリオ3: 職場メンバーの楽曲ドラフト
- **参加者**: 8人の同僚
- **対象**: 今年リリースされた楽曲
- **時期**: 年末
- **フロー**:
  1. 昼休みにサクッと実施
  2. 各自のお気に入り楽曲を選択
  3. 後日プレイリスト作成

### 効果的なマーケティングチャネル

| チャネル | 効果 | OnlineDraftとの相性 | 施策例 |
|----------|------|---------------------|--------|
| **X (Twitter)** | エンタメ・ゲーム系は盛り上がりやすい | ⭐⭐⭐ 最高 | ハッシュタグ文化、結果シェア |
| **Discord** | ゲームコミュニティの本拠地 | ⭐⭐⭐ 最高 | Bot連携、サーバーイベント |

| **TikTok** | 若年層へのリーチ | ⭐⭐ 良い | 短尺プレイ動画 |
| **インフルエンサー** | 配信者とのコラボで爆発力 | ⭐⭐⭐ 最高 | リスナー参加企画 |

### Z世代の行動特性
- 半数超が「暇つぶしにSNSをチェック」
- 3割近くが「ゲームをする」
- SNSとゲームアプリを行き来する傾向

### マーケティングのポイント

#### 1. 「楽しそう」と思わせる
- ゲーム要素（スロット演出等）が強みになる
- プレイ動画・スクリーンショットで視覚的訴求

#### 2. シェアしやすい仕組み
- URLシェア機能は既に実装済み
- 結果画面のSNSシェアボタン追加を検討

#### 3. コミュニティとの相性
- Discord連携は超有効
- 配信者がリスナーと一緒にドラフトするシーンは映える

### 具体的な施策案

| 施策 | 詳細 | 優先度 |
|------|------|--------|

| Discord Bot | ルーム作成・参加をDiscord上で完結 | 中 |

## 8. ユーザーの声分析（競合の不満点）

### 競合サービスへの不満

| 不満 | 詳細 | OnlineDraftの対応状況 |
|------|------|----------------------|
| **スマホ対応が不十分** | PCでしか使いづらい | ✅ 完全レスポンシブ対応済み |
| **複数ツール併用が面倒** | スプレッドシート+LINE+あみだくじ | ✅ 全部1つで完結 |
| **順番待ちが長い** | 8人で回すと自分の番まで暇 | ✅ 一斉入力で解決 |
| **演出がない** | 淡々と進んで盛り上がらない | ✅ スロット演出あり |
| **POG専用で汎用性がない** | 野球やアニメでも使いたい | ✅ 何でもOK |
| **通信エラーで重複登録** | Firebaseの不具合報告あり | ⚠️ 要テスト確認 |

### 初心者ユーザーの困りごと

| 困りごと | OnlineDraftでの解決策 |
|----------|----------------------|
| ドラフト中盤でリストが弾切れ | チャットで相談しながら進められる |
| 指名順位を決めるのに時間がかかる | 時間制限なしで自分のペースOK |
| ツールの使い方がわからない | 直感的なUI、ガイド機能あり |

### 競合ツールの具体的な問題点

#### 自作ツール系の問題（sanshonokiの日記より）
- スマホ対応含めてデザインに問題
- 通信環境の問題でFirebaseに重複登録される不具合
- GoogleForms + LINEあみだくじ等、複数ツール併用が必要

#### POG専用ツールの問題
- POG以外（野球、アニメ等）で使えない
- データベース連携が逆に制約になることも
- アカウント登録必須で気軽さがない

## 9. OnlineDraftの強み・差別化ポイント

### コアバリュー

| 強み | 説明 |
|------|------|
| **汎用性** | 野球・競馬・アニメ・音楽なんでもOK！ジャンル縛りなし |
| **一斉入力方式** | 順番待ちなし！全員同時に選んで即開票 |
| **演出の楽しさ** | スロット風開票演出で盛り上がる！ |
| **登録不要** | URLシェアだけで即参加、匿名OK |
| **音声通話前提** | Discord/LINEと併用で最高の体験 |

### 競合との差別化マトリクス

```
                    汎用性
                      ↑
                      |
     OnlineDraft ★   |
                      |
    ───────────────────────────→ 気軽さ
                      |
         POG専用ツール |   プロ野球#LIVE
                      |
                      ↓
```

### 刺さるポイント

> **「既存サービスに満足できない人」に刺さる！**

| ターゲット | ニーズ |
|------------|--------|
| POG限定ツールユーザー | 汎用性がほしい |
| 順番制ドラフトユーザー | 待ち時間イヤ |
| アカウント登録嫌い | 手軽に始めたい |
| スプレッドシート管理者 | 演出で盛り上がりたい |

### TOPページで訴求すべきポイント

1. **汎用性**: 「野球も、アニメも、なんでもOK！」
2. **一斉入力**: 「順番待ちなし！全員同時にドラフト」
3. **演出**: 「スロット風開票で盛り上がる！」
4. **気軽さ**: 「登録不要、URLだけで即参加」
5. **スマホ対応**: 「スマホでもPCでも快適プレイ」
