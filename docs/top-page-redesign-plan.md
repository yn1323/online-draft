## 1. 概要

### 1.1 リニューアルの目的
- サービスの価値・強みを明確に伝える
- AIO（AI Overview）対策で検索エンジン・AI検索に最適化
- ユーザーの行動喚起（CTA）を強化

### 1.2 現状の課題
| 課題 | 詳細 |
|------|------|
| 競合との差別化が不明確 | 何が他と違うかわからない |
| 機能説明が簡潔すぎる | 何ができるか伝わりにくい |
| 利用シーンがイメージしにくい | テキストのみで具体性に欠ける |
| AIO対策がされていない | AIが引用しにくい構造 |

### 1.3 リニューアル後のゴール
- **ユーザー**: 5秒でサービス概要を理解できる
- **AIO**: FAQ形式でAIが引用しやすい構造
- **コンバージョン**: CTAが複数箇所に配置され行動を促す

## 2. AIO対策の方針

### 2.1 AIOとは
AIO（AI Optimization）は、GoogleのAI Overview等のAI検索で自社サイトが引用されることを目的とした施策。

参考: [2025年AIO対策完全ガイド](https://allweb-consulting.co.jp/information/aio/)

### 2.2 対策ポイント

| ポイント | 実装方法 |
|----------|----------|
| **明確な情報構造** | FAQ形式でQ&Aを記述 |
| **具体的なファクト** | 「2-8人」「10-15分」「完全無料」など数値を明記 |
| **質問意図への回答** | 「何ができるか」「どう使うか」を明確に |
| **指名検索の促進** | 「オンラインドラフト会議」をH1で強調 |

### 2.3 構造化データ（将来対応）
- JSON-LD形式でFAQPageスキーマを追加
- WebApplicationスキーマの追加

## 3. セクション構成

### 3.1 全体構成（7セクション）

```
┌─────────────────────────────────────┐
│  Section 1: ヒーローエリア          │  ← ファーストビュー
│  - タイトル、キャッチコピー、CTA    │
├─────────────────────────────────────┤
│  Section 2: サービス説明            │  ← What is this?
│  - 概要説明、3つの特徴カード        │
├─────────────────────────────────────┤
│  Section 3: 差別化ポイント          │  ← Why us?
│  - 4つの強み（チェックリスト形式）  │
├─────────────────────────────────────┤
│  Section 4: 使い方                  │  ← How to use?
│  - 3ステップ、CTA                   │
├─────────────────────────────────────┤
│  Section 5: ユースケース            │  ← Who uses this?
│  - 4つの利用シーンカード            │
├─────────────────────────────────────┤
│  Section 6: FAQ                     │  ← AIO対策の核心
│  - 5つのQ&A                         │
├─────────────────────────────────────┤
│  Section 7: フッターCTA             │  ← 最後の行動喚起
│  - CTA、コピーライト                │
└─────────────────────────────────────┘
```

### 3.2 情報の優先順位

| 優先度 | セクション | 理由 |
|--------|------------|------|
| 1 | ヒーロー | 第一印象、離脱防止 |
| 2 | サービス説明 | 何ができるか理解 |
| 3 | 差別化 | 競合との違いを訴求 |
| 4 | FAQ | AIO対策、疑問解消 |
| 5 | 使い方 | 利用ハードル低減 |
| 6 | ユースケース | 具体的なイメージ |
| 7 | フッターCTA | 最後のプッシュ |


### 7.2 各ファイルの役割

| ファイル | 役割 |
|----------|------|
| `index.tsx` | 全セクションを統合、ルーム作成/参加ロジック |
| `HeroSection.tsx` | H1、キャッチコピー、CTA、ルーム参加フォーム |
| `ServiceSection.tsx` | サービス概要、3つの特徴カード |
| `DifferentiationSection.tsx` | 4つの差別化ポイント |
| `HowToUseSection.tsx` | 3ステップ、中間CTA |
| `UseCaseSection.tsx` | 4つのユースケースカード |
| `FAQSection.tsx` | 5つのQ&A、Accordion |
| `FooterCTASection.tsx` | 最終CTA、コピーライト |

```tsx
type HowToUseSectionProps = {
  onCreateRoom: () => void;

  onCreateRoom: () => void;
};
```
